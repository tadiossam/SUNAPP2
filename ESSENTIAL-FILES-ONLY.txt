WINDOWS SETUP FILES (5 files - all you need!):
----------------------------------------------
1. START-WINDOWS.bat
   → Run this to start the server
   → Use this EVERY TIME you want to run the app

2. SETUP-FIRST-TIME.bat
   → Run this ONCE after downloading
   → Installs dependencies and sets up environment

3. ALLOW-FIREWALL.bat
   → Run this ONCE as Administrator
   → Allows network access from other computers

4. CHECK-IP.bat
   → Check if your IP is 192.168.0.34
   → Run if other computers can't connect

5. WINDOWS-README.txt
   → Complete user guide
   → Read this for help
